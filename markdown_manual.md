markdown的标题有两种方式来标注

### 列表

Markdown 支持有序列表和无序列表.

无序列表使用星号, 加号, 和连字符 -- 这些符号是可互换的 -- 最为列表标记:

```
*   Red
*   Green
*   Blue
```

等价于:

```
+   Red
+   Green
+   Blue
```

以及:

```
-   Red
-   Green
-   Blue
```

有序列表使用数字加句号:
```
1.  Bird
2.  McHale
3.  Parish
```

需要注意的是这里的数字序号对于最终生成 HTML 是没有影响的. 这里 Markdown 输出的 HTML 列表是:
```
<ol>
<li>Bird</li>
<li>McHale</li>
<li>Parish</li>
</ol>
```
即使你把列表写成这样:
```
1.  Bird
1.  McHale
1.  Parish
```

甚至这样:
```
3. Bird
1. McHale
8. Parish
```

### 分隔符
如果一行中只有三个以上的连字符, 星号, 或者下划线则会在该位置生成一个 `<hr /> `标签. 星号和连字符之间的空格也是允许的. 下面的例子都会生成一条水平线:

```
* * *

***

*****

- - -

---------------------------------------
```

### 链接

markdown的链接的表示方法主要分为行内式和参考式两种

**行内式**

```
文字链接 [链接名称](http://链接网址 "title") 
自动链接 <http://链接网址>
```

其中，文字链接中的title可写可不写，title属性的效果是鼠标悬停在链接上会出现指定的title文字

**参考式**

```
这个链接用 1 作为网址变量 [Google][1].
这个链接用 yahoo 作为网址变量 [Yahoo!][yahoo].
然后在文档的结尾为变量赋值（网址）
  [1]: http://www.google.com/
  [yahoo]: http://www.yahoo.com/
```

### 锚点

锚点就是页内超链接，也就是链接到本文档内部的某些元素上，实现当前页面中的跳转。

**描述**

在准备跳转到的指定标题后插入锚点{\#标记}，然后在文档的其他地方写上连接到锚点的链接

```
1. ## 0. 目录{#index}
2.
3. 跳转到[目录](#index)
```

### 图片

跟链接的方法区别在于前面加了个感叹号 !
```
![图片名称](http://图片网址)
```
也可以像网址那样对图片网址使用变量

这个链接用 1 作为网址变量 [Google][1].
然后在文档的结尾位变量赋值（网址）
```
 [1]: https://www.google.com/logo.png
 ```
也可以使用 HTML 的图片语法来自定义图片的宽高大小
```html
<img src="htt://example.com/sample.png" width="400" height="100">
```

### 表格

**语法说明**

1. 第一行为表头，第二行分隔表头和主题部分，第三行开始为一个表格行
2. 列之间用\|分隔
3. 第二行还可以指定不同列的对齐方式

对齐方式|说明
----|----
\-\-\-|居中对齐
:\-\-\-|靠左对齐
\-\-\-:|靠右对齐

代码：
```
学号|姓名|分数
-|-|-
小明|男|75
小红|女|79
小陆|男|92
```

显示效果:

学号|姓名|分数
-|-|-
小明|男|75
小红|女|79
小陆|男|92
